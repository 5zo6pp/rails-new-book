# rails-new-book
Railsのbookアプリを一から作る (2016-11-22)

## Vagrantの起動

    $ vagrant up
    $ vagrant ssh

## このリポジトリのclone

事前にこのリポジトリをForkすること！

    [vagrant@jessie: ~] $ cd /vagrant
    [vagrant@jessie: vagrant] $ git clone https://github.com/（自分のアカウント名）/rails-new-book.git
    [vagrant@jessie: vagrant] $ cd rails-new-book
    [vagrant@jessie: vagrant] $
